# Per-Chapter Guide Card Template

> 引路卡不是设定卡复述。它只回答：**这一章怎样才好看**。
> **Sanitized version** — 具体角色名、世界设定术语、物件 name 抽象化；模板结构 + craft 维度保留。

---

## 使用原则

- **每章单独写**，不复用上一章
- **写功能，不写可复制的正文句子**
- 允许列关键台词，但正文旁白不要从引路卡照抄
- 引路卡放在 `_通讯/ChXX/大纲.md` 顶部或单独文件 `_通讯/ChXX/引路卡.md`

---

## 模板

```md
# ChXX《章名》引路卡

## 当章事件

这一章外部剧情真正发生什么。
**只写本章能发生的**，不提前透后章底牌。

## 关系动作

[Character A] 和 [Character B] 之间发生了什么变化：
> 隐瞒 / 拆穿 / 交还 / 认领 / 等待 / 放手 / 共担 / 反制 / 拒绝 / 允许

(每个 verb 都是关系动力学的具体 maneuver — 不要泛泛"两人关系推进了")

## 控制味（CP 张力 specifics）

本章**具体是哪一种控制**，不写泛泛"占有"：

- 检查 / 记录 / 压味
- 等待 / 交还 / 认领
- 拒绝解除 / 命的处置权
- 锚定 / 放手
- (项目自定 vocabulary)

说明这套控制**怎样进入场面**（不是哪句话讲了控制，是哪个动作让读者感到控制）。

## 身体入口

本章**最该写哪些身体细节**。不要泛用词库，要**贴合剧情**。

举例 pattern（具体内容因项目而异）：
- 掌心被 [object] 硌出 [physical mark]
- [身体动作] 后 [body part] 承力迟一拍
- [internal sensation] 被 [event] 余震牵动
- [某 sensitive area] 因为说谎先乱
- [bone / joint] 被扣住，脉搏被摸到

→ 每条都是 specific 的（身体部位 + 触发事件 + 反应），不是泛"她抖了一下"。

## 可写边界

本章可以写到哪里：

- 贴近 / 扣腕 / 压掌 / 喘息 / 汗 / 红痕 / 血 / 眼尾水光 / 腿软 / 下腹牵痛 / 某 sensitive area 失控
- (按当章实际情况增删)

## 禁区

本章**不能写什么**：

- 不接吻 / 不脱衣 / 不直接性行为
- 不直白告白
- 不提前写后章真相
- (按当章实际情况增删)

## 关键台词

列本章**必须保留的刀口台词**。

这些可以来自 reviewer AI 骨架，也可以由 writer AI 改写。

注意：**关键台词可保留，旁白不照抄"标注腔"**。

## 语感种子

贴 1 段已有正文或样章，**用来校准这一章的语感**。

说明为什么选它：节奏 / 感官 / 物件 / 台词 / 视角，具体是哪一点。

## 结尾锁扣

这一章**最后用什么动作 / 物件 / 短句扣住**。

举例 pattern：
- [object] 回到 [body part]（物件归位）
- [object] 递下 [scene]（物件移交）
- 某 [physical boundary] 关闭（空间断点）
- 某句话**没有说完**（语音断点）

→ 每章一个明确的"扣"，让读者带着具体物件 / 动作记忆进入下一章。
```

## 设计原理（为什么不复用）

| 设计 | 为什么 |
|---|---|
| 引路卡 ≠ 设定卡 | 设定卡是世界观 reference（不变）；引路卡是**这一章的写作设计**（每章重设） |
| 写功能 ≠ 写句子 | 模板告诉你这一章"需要让读者感到什么"，**不告诉你具体怎么写**——句子由 writer AI 现场长出来 |
| 关键台词列表 + 旁白不抄 | 区分**钉子**（保留）vs **填充**（重写）—— 避免 writer 把 reviewer 的骨架描述当正文写 |
| 控制味每章不同 | 防止角色关系动力学循环单一 maneuver |

## 学术对照

| 概念 | 学术 anchor |
|---|---|
| 引路卡作为 plan-as-resource 而非 plan-as-prescription | Suchman (1987) *Plans and Situated Actions* |
| Per-chapter customization > 通用模板 | situated learning — 工艺不能从 abstract template 完全 transfer，需要 case-specific scaffolding |
